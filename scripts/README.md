# Scripts Directory

Utility scripts for development, testing, and maintenance tasks.

## Contents

### Python Scripts

- **`query_constraint.py`** - Query database constraints
- **`run_fix.py`** - Run database fixes

### Batch Scripts

- **`dev-wt.bat`** - Development startup with Windows Terminal

**Note:** Main startup scripts (`start.bat`, `dev.bat`) are kept in the project root for easy access.

## Usage

Run Python scripts from the project root:

```bash
python scripts/query_constraint.py
```

Run batch scripts directly:

```bash
scripts\dev-wt.bat
```
